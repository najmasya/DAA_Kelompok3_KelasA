# Perbandingan Algoritma Unbounded Knapsack
Proyek ini membandingkan performa (kecepatan dan akurasi) dari tiga algoritma untuk menyelesaikan Unbounded Knapsack Problem: Greedy by Profit, Greedy by Density, dan Dynamic Programming.

# Struktur Proyek
├── run.ipynb # Notebook utama berisi semua kode, analisis, dan visualisasi

├── Data_Anteraja.csv # Dataset yang digunakan untuk pengujian performa algoritma

└── README.md


# Kesimpulan:
-   Dynamic Programming memberikan solusi sempurna tetapi dengan waktu komputasi jauh lebih lambat.
-   Algoritma Greedy jauh lebih cepat, tetapi mengorbankan akurasi. Greedy-Density memberikan keseimbangan yang lebih baik dibandingkan Greedy-Profit untuk dataset ini.
